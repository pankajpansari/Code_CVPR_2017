CODE FOR TRUNCATED MAX-OF-CONVEX MODELS

ABOUT
=====
This is an implementation of the range expansion algorithm for energy minimization in truncated max-of-convex models, as described in [1].

AUTHOR(S)
=========
Pankaj Pansari

INSTALLATION
============
Download the code folder and run make

USAGE
=====
Usage is similar to the example shown in 'example.cpp'.

==========
